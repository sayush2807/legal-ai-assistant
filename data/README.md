# Data

Raw dataset files are not stored in this repository. All notebooks download data automatically via the HuggingFace `datasets` library on first run.

## CaseHOLD (LexGLUE)

The canonical dataset used across all notebooks.

```python
from datasets import load_dataset
dataset = load_dataset("coastalcph/lex_glue", "case_hold")
```

| Split      | Examples |
|------------|----------|
| Train      | 45,000   |
| Validation | 3,900    |
| Test       | 3,600    |

Each example contains a legal case excerpt (`context`) and five candidate holdings (`endings`), one of which is correct (`label`). The task is to identify the correct holding.

**HuggingFace page:** https://huggingface.co/datasets/coastalcph/lex_glue

## Fine-Tuning Dataset

`finetune_casehold.jsonl` is derived from the CaseHOLD train split and is excluded from this repo (83 MB). It is regenerated by running `notebooks/01_qlora_finetune.ipynb` from the raw dataset.
